---

## RBAC 層級定義

| Level | 名稱 | 說明 |
|:------|:-----|:-----|
| 0 | Guest | 未登入訪客 |
| 1 | Viewer | 唯讀檢視 |
| 2 | Editor | 新增/編輯 |
| 3 | Manager | 管理/審批 |
| 4 | Admin | 系統管理 |
| 5 | Owner/Super | 超級管理員 |
